CATEGORY 2 · CONTINUITY & DIFFERENTIABILITY

|x| Continuity Proof at x = 0

The foundational proof that separates genuine understanding from rote memorisation — demonstrate that the absolute value function is continuous at the origin using left‑hand and right‑hand limits.

f(x) = |x| · The graph approaches (0,0) from both sides without any break or jump.

📐 The Formal Proof (Limit Definition)

A function f(x) is continuous at x = a if and only if:

limx→a f(x) = f(a)

This means three things must hold:

  1. The limit as x approaches a exists (both left‑hand and right‑hand limits are equal).
  2. The function is defined at a (f(a) exists).
  3. The limit equals the function value.

Step 1: Left‑hand limit (x → 0⁻)

For x < 0, |x| = -x. Therefore,

limx→0⁻ |x| = limx→0⁻ (-x) = 0

Step 2: Right‑hand limit (x → 0⁺)

For x > 0, |x| = x. Therefore,

limx→0⁺ |x| = limx→0⁺ (x) = 0

Step 3: Function value at x = 0

f(0) = |0| = 0

Conclusion

Since LHL (0) = RHL (0) = f(0) = 0, all three conditions are satisfied. Therefore, |x| is continuous at x = 0.

This proof is the gateway to understanding that continuity alone does not guarantee differentiability — the next module will show that at the very same point, |x| fails to be differentiable because the slopes from the left and right disagree.

🔬 Epsilon‑Delta Proof (Optional, but Impressive)

For any ε > 0, we need to find δ > 0 such that | |x| - 0 | < ε whenever 0 < |x - 0| < δ.

Observe that | |x| | = |x|. So we need |x| < ε whenever |x| < δ. Simply choose δ = ε. Then whenever |x| < δ, we have |x| < ε, which implies | |x| - 0 | < ε.

Thus, by the ε-δ definition of a limit, limx→0 |x| = 0 = f(0). This proves continuity at x = 0 in the most rigorous form — and the choice δ = ε is beautifully simple.

🎙️ Interview Narration Script

"To prove |x| is continuous at zero, I first recall the definition: a function is continuous at a point if the left‑hand limit, the right‑hand limit, and the function value all exist and are equal. For |x|, when x is negative, the function behaves as -x, so the left‑hand limit approaching zero is zero. When x is positive, |x| is simply x, so the right‑hand limit is also zero. And at x equals zero, the function value is zero. Since all three quantities match, the function is continuous. This is the same logic that later shows |x| is not differentiable at zero — continuity is about the values matching, differentiability is about the slopes matching."

🎯 Likely Interview Questions

Q1: Prove that |x| is continuous at x = 0.+

Left‑hand limit: limx→0⁻ |x| = limx→0⁻ (-x) = 0. Right‑hand limit: limx→0⁺ |x| = limx→0⁺ x = 0. Function value: f(0) = 0. Since LHL = RHL = f(0), the function is continuous at x = 0. I can also provide the epsilon‑delta proof if you'd like.

Q2: What is the formal definition of continuity?+

A function f(x) is continuous at x = a if limx→a f(x) = f(a). This requires the limit to exist, the function to be defined at a, and the two to be equal. In epsilon‑delta terms: for every ε>0 there exists δ>0 such that |f(x)−f(a)|<ε whenever |x−a|<δ.

Q3: Why is |x| not differentiable at x = 0 even though it is continuous?+

Continuity only guarantees the function values match at the point. Differentiability requires the slopes to match. For |x|, the left‑hand derivative is -1 and the right‑hand derivative is +1, so the slopes disagree, making it non‑differentiable despite being perfectly continuous. |x| is the classic counterexample to "continuous implies differentiable".

Q4: Can you give the epsilon‑delta proof for |x| at x = 0?+

We need to show: for any ε>0, we can find δ>0 such that ||x|−0|<ε whenever 0<|x−0|<δ. Since ||x|| = |x|, the condition becomes |x|<ε. So simply pick δ = ε. Then |x|<δ implies |x|<ε, which satisfies the definition. Therefore, limx→0 |x| = 0 = f(0).

Q5: How would you visualise continuity at x = 0 on the graph of |x|?+

The graph of |x| is a V‑shape with the vertex at the origin. There is no break, jump, or hole at x = 0 — the two line segments meet perfectly at the point (0,0). That visual seamlessness is continuity. The sharp corner at that same point, however, indicates the absence of a unique tangent, hence no differentiability.

The proof is simple but foundational: it establishes that |x| has no break at the origin, even though we later learn it has a sharp corner that makes it non‑differentiable.
