|x| — Absolute Value Function

The V‑shaped modulus function: piecewise definition, sharp corner at the origin, and the classic interview proof that it is continuous but not differentiable at x = 0.

f(x) = |x| · V‑shape · Continuous everywhere · Not differentiable at x = 0

🧩 Piecewise Definition & Anchor Points

DEFINITION

|x| = { x  if x ≥ 0
         { -x  if x < 0

VERTEX

(0, 0) — the sharp corner

LEFT ARM (x < 0)

Line y = -x, slope = -1

RIGHT ARM (x > 0)

Line y = x, slope = +1

Narration while drawing: "For x greater than or equal to zero, |x| is simply x — a straight line through the origin with slope 1. For x less than zero, |x| flips the sign and becomes -x, which is a straight line with slope -1. The two lines meet at the origin, forming a perfect V‑shape. Crucially, there is a sharp corner at (0,0) — the left side approaches with slope -1, the right side with slope +1, and these two slopes do not match. This geometric fact is the visual reason why |x| is not differentiable at x = 0, even though it is perfectly continuous there."

📐 The Classic Proof: Continuous but NOT Differentiable at x = 0

Part 1: Continuity at x = 0 ✓

Left‑hand limit: limx→0⁻ |x| = limx→0⁻ (-x) = 0
Right‑hand limit: limx→0⁺ |x| = limx→0⁺ (x) = 0
Function value: f(0) = |0| = 0
Conclusion: LHL = RHL = f(0) = 0, so |x| is continuous at x = 0.

Part 2: Differentiability at x = 0 ✗

Left‑hand derivative: limh→0⁻ [|0+h| - |0|] / h = limh→0⁻ |h|/h = limh→0⁻ (-h)/h = -1
Right‑hand derivative: limh→0⁺ [|0+h| - |0|] / h = limh→0⁺ |h|/h = limh→0⁺ h/h = +1
Conclusion: LHD (-1) ≠ RHD (+1), so the derivative does not exist at x = 0. |x| is not differentiable at x = 0.

Key insight: Continuity is a necessary but not sufficient condition for differentiability. |x| is the canonical example that proves this distinction — a function can be perfectly unbroken (continuous) yet have a sharp corner where no unique tangent line can be drawn (non‑differentiable).

🎯 Likely Interview Questions

Q1: Draw the graph of |x| and explain its shape. +

The graph is a V‑shape meeting at the origin. For x ≥ 0 it's the line y = x with slope +1; for x < 0 it's the line y = -x with slope -1. The sharp corner at (0,0) is the defining feature — it's where the function transitions from negative slope to positive slope abruptly.

Q2: Is |x| continuous at x = 0? Prove it. +

Yes. LHL as x→0⁻ of (-x) = 0. RHL as x→0⁺ of x = 0. f(0) = 0. Since all three are equal, |x| is continuous at x = 0. I can write this proof in under a minute.

Q3: Is |x| differentiable at x = 0? Prove it. +

No. LHD = limh→0⁻ |h|/h = limh→0⁻ (-h)/h = -1. RHD = limh→0⁺ |h|/h = limh→0⁺ h/h = +1. Since LHD ≠ RHD, the derivative does not exist. The sharp corner at the origin prevents a unique tangent line.

Q4: What does the non‑differentiability of |x| tell us about continuity and differentiability? +

It proves that continuity is necessary but not sufficient for differentiability. A function can be perfectly continuous — no breaks, no jumps — yet still lack a derivative at a point. |x| is the simplest, most elegant demonstration of this crucial distinction.

Q5: What is the derivative of |x| for x ≠ 0? +

For x > 0, the derivative is +1. For x < 0, the derivative is -1. This is often written as f'(x) = x/|x| for x ≠ 0, or simply as the signum function: f'(x) = sign(x) where sign(x) = 1 for x>0 and -1 for x<0.

Q7: How would the graph of |x - 3| differ from |x|? +

It shifts the entire V‑shape 3 units to the right. The vertex moves from (0,0) to (3,0). The left arm (x < 3) has slope -1, the right arm (x > 3) has slope +1. The shape remains identical; only the location of the sharp corner changes.
