CATEGORY 1 · GRAPHS

y = 50 - |x| (with y ≥ 0)

The classic interview triangle problem: building a constrained absolute value graph, verifying area via both geometry and integration, and demonstrating layered transformation thinking.

y = 50 - |x| , y ≥ 0 · Triangle with vertices (-50,0), (0,50), (50,0)

🧱 Step‑by‑Step Construction

Step 1 — Base graph: Start with y = |x|. This is the V‑shape with vertex at (0,0), slope -1 for x<0 and +1 for x>0.

Step 2 — Reflection: y = -|x|. This flips the V upside down, producing an inverted V with vertex still at (0,0), now opening downward. Slopes become +1 (left) and -1 (right).

Step 3 — Vertical shift: y = 50 - |x|. Adding 50 shifts the entire graph upward by 50 units. The vertex moves from (0,0) to (0,50). The arms now extend downward from the vertex.

Step 4 — Domain restriction: The condition y ≥ 0 cuts off any part of the graph below the x‑axis. Solve 50 - |x| = 0 → |x| = 50 → x = ±50. So the graph exists only for x ∈ [-50, 50], forming a closed triangle.

📐 Geometric Properties

VERTICES

A(-50, 0), B(0, 50), C(50, 0)

BASE LENGTH

100 units (from x=-50 to x=50)

HEIGHT

50 units (y‑coordinate of vertex)

AREA

½ × base × height = ½ × 100 × 50 = 2500 sq units

Slope analysis: For x < 0, y = 50 - (-x) = 50 + x → slope = +1. For x > 0, y = 50 - x → slope = -1. The vertex at (0,50) is a sharp corner where the slope changes abruptly from +1 to -1 — so the function is not differentiable at x = 0.

🧮 Area Verification via Integration

The area can be computed as the definite integral of (50 - |x|) from -50 to 50. Since the function is piecewise, we split at x = 0:

Area = ∫-500 (50 + x) dx  +  ∫050 (50 - x) dx

Left integral: ∫(50 + x)dx = 50x + x²/2. Evaluated from -50 to 0: (0 + 0) - ( -2500 + 1250) = 0 - (-1250) = 1250.
Right integral: ∫(50 - x)dx = 50x - x²/2. Evaluated from 0 to 50: (2500 - 1250) - 0 = 1250.
Total: 1250 + 1250 = 2500, matching the geometric formula.

Interview Narration: "I can confirm this area in two completely independent ways — the geometric formula for a triangle and the definite integral. The fact that they match confirms both the graphical analysis and the integration. This dual verification is exactly what a panelist wants to see: understanding, not just computation."

🎯 Likely Interview Questions

Q1: Draw the graph of y = 50 - |x| with y ≥ 0. Explain each transformation.+

Start with |x| (V at origin). Reflect to -|x| (inverted V). Shift up by 50 → vertex at (0,50). The condition y ≥ 0 restricts the domain to [-50,50]. The resulting graph is an isosceles triangle. I narrate each transformation while drawing to demonstrate layered thinking.

Q2: What is the area under this graph? Show two methods.+

Method 1 (Geometry): Triangle with base 100, height 50 → area = ½ × 100 × 50 = 2500.
Method 2 (Integration): Split at x=0. ∫-500 (50+x)dx + ∫050 (50-x)dx = 1250 + 1250 = 2500. Both methods agree.

Q3: Is the function differentiable at x = 0? Why?+

No. For x<0, f(x)=50+x → derivative +1. For x>0, f(x)=50-x → derivative -1. The left‑hand derivative is +1, the right‑hand derivative is -1. They do not match, so the function is not differentiable at x=0. The sharp vertex confirms this visually.

Q4: How would the graph change if the condition was y ≥ 10 instead of y ≥ 0?+

We would clip the graph at y = 10, creating a truncated triangle — a trapezoid. The new x‑intercepts: solve 50 - |x| = 10 → |x| = 40 → x = ±40. The graph would be a trapezoid with top base from (-40,10) to (40,10) and bottom base from (-50,0) to (50,0). Area would be that of the original triangle minus the small triangle on top.

Q5: What real‑world situation could this graph represent?+

A water reservoir with a triangular cross‑section: depth 50m at center, width 100m at top, with the water surface at y=0. Or a cost function that decreases linearly as you move away from a central point, until it reaches zero. The absolute value models symmetric penalties or distances.

Q6: How would you handle the integral if the function was y = |50 - |x||?+

That would be a W‑shaped graph. For x between -50 and 50, 50 - |x| is already ≥ 0, so the absolute value does nothing. Outside that interval, 50 - |x| becomes negative, and the absolute value would flip it upward, creating additional triangular regions. The integral would need to be split at ±50 and handled piecewise.
