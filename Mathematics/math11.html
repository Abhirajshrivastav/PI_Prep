CATEGORY 2 · CONTINUITY & DIFFERENTIABILITY

|x| Differentiability Proof at x = 0

The sharp corner: prove that the absolute value function fails to be differentiable at the origin because the left‑hand and right‑hand derivatives disagree.

f(x) = |x| · Continuous at 0, but the slopes from left and right differ → not differentiable

📐 The Formal Proof (Definition of Derivative)

The derivative of f(x) at x = a is defined as:

f'(a) = limh→0 [f(a+h) − f(a)] / h

if and only if this limit exists and is the same from both sides. For |x| at x = 0:

Left‑hand derivative (h → 0⁻)

For h < 0, |h| = -h. So:

LHD = limh→0⁻ [|0+h| − |0|] / h
    = limh→0⁻ |h| / h
    = limh→0⁻ (-h) / h
    = limh→0⁻ -1
    = -1

Right‑hand derivative (h → 0⁺)

For h > 0, |h| = h. So:

RHD = limh→0⁺ [|0+h| − |0|] / h
    = limh→0⁺ |h| / h
    = limh→0⁺ h / h
    = limh→0⁺ 1
    = +1

Conclusion

Since LHD (-1) ≠ RHD (+1), the derivative at x = 0 does not exist. The absolute value function is not differentiable at the origin.

Together with the continuity proof, |x| demonstrates the classic result: continuity is necessary but not sufficient for differentiability.

🔍 Visual Intuition — The Sharp Corner

The graph of |x| is a V‑shape. At x = 0, the left arm comes in with slope -1, and the right arm departs with slope +1. There is no single tangent line that can be drawn at the vertex — any line through (0,0) would either cut across the graph or fail to match both sides. This geometric failure is exactly what the derivative calculation captures: the limit of the secant slope differs depending on which side you approach from.

Narration: "If I try to draw a tangent at the sharp corner, I can't — there are two equally valid tangent lines trying to exist, one with slope -1 and one with slope +1. Differentiability requires exactly one tangent line. The failure of uniqueness is why the derivative does not exist."

🎯 Likely Interview Questions

Q1: Prove that |x| is not differentiable at x = 0.+

Compute the left‑hand derivative: for h<0, |h|/h = -h/h = -1. So LHD = -1. Right‑hand derivative: for h>0, |h|/h = h/h = +1. So RHD = +1. Since LHD ≠ RHD, the derivative does not exist. |x| is not differentiable at x = 0.

Q2: What is the geometrical meaning of this non‑differentiability?+

The graph has a sharp corner at the origin. The slope of the curve changes abruptly from -1 to +1 without any smooth transition. There is no unique tangent line at that point — two different lines with slopes -1 and +1 both fit the curve perfectly on their respective sides, so the derivative cannot be defined as a single number.

Q3: How does this proof connect to the continuity proof from the previous module?+

The continuity proof showed that the function values match at x=0 (LHL = RHL = f(0) = 0). The differentiability proof shows that the slopes do not match (LHD = -1, RHD = +1). Together they demonstrate that continuity is necessary but not sufficient for differentiability — a function can be perfectly unbroken yet still have a sharp corner that prevents a unique derivative.

Q4: Give another example of a function that is continuous but not differentiable at a point.+

x^(2/3) at x=0. The function is continuous (0^0 = 0, and both limits approach 0), but the derivative blows up to infinity (vertical tangent), so it is not differentiable. Also, any function with a cusp, like the Weierstrass function (fractal, nowhere differentiable), though that's more advanced. The simplest classroom example remains |x|.

Q5: What is the derivative of |x| for x ≠ 0?+

For x > 0, |x| = x, so derivative is +1. For x < 0, |x| = -x, so derivative is -1. This is often written compactly as f'(x) = x/|x| for x ≠ 0, or as the signum function: sign(x) where sign(x)=1 for x>0 and -1 for x<0.

⚡ 30‑SECOND INTERVIEW SUMMARY

"The absolute value function |x| is not differentiable at x = 0 because the left‑hand derivative equals -1 while the right‑hand derivative equals +1. These two limits disagree, so the derivative — which requires a unique limit — does not exist at that point. Geometrically, this corresponds to the sharp corner at the vertex of the V‑shaped graph, where no single tangent line can be drawn. Combined with the earlier proof that |x| is continuous at the same point, this is the definitive classroom example that continuity does not imply differentiability. Every interview panelist expects you to understand this distinction and be able to prove it clearly."
