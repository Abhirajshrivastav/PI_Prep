log x & |log x| — Graphs

The natural logarithm curve and its absolute value: asymptotic behavior, the crucial point (1,0), and the sharp corner that introduces the concept of non‑differentiability for transformed functions.

f(x) = ln x · Domain (0,∞)

f(x) = |ln x| · V‑shape at x=1

🔑 Key Features & Narration

VERTICAL ASYMPTOTE

x = 0 (ln x → -∞ as x→0⁺)

X‑INTERCEPT

(1, 0) — ln 1 = 0

KEY POINTS

(e, 1), (1/e, -1)

DERIVATIVE

1/x (always positive → strictly increasing)

Narration for ln x: "The natural logarithm function is defined only for positive x. As x approaches zero from the right, the curve dives toward negative infinity — this is a vertical asymptote at x = 0. The graph crosses the x‑axis at (1, 0) because ln 1 = 0. After that, it rises slowly, passing through (e, 1). The slope is 1/x, so it's steep near zero but flattens as x grows — the function is always increasing but at a decreasing rate."

Narration for |ln x|: "To get |ln x|, we reflect the negative parts of the curve above the x‑axis. For x between 0 and 1, ln x is negative, so its absolute value becomes -ln x — a decreasing curve that falls from positive infinity at x=0⁺ down to 0 at x=1. For x greater than 1, |ln x| is identical to ln x, rising slowly toward infinity. The two pieces meet at (1,0), forming a sharp corner. This means |ln x| is continuous everywhere on its domain but not differentiable at x = 1."

⚠️ The Sharp Corner at x = 1

For x < 1, |ln x| = -ln x → derivative = -1/x. As x → 1⁻, derivative → -1. For x > 1, |ln x| = ln x → derivative = 1/x. As x → 1⁺, derivative → +1. The left‑hand derivative is -1, the right‑hand derivative is +1. They do not match, so |ln x| is not differentiable at x = 1, even though it's perfectly continuous there. This is another classic example — like |x| — of a function that is continuous but not differentiable due to an abrupt change in slope.

🎯 Likely Interview Questions

Q1: Draw the graph of ln x and explain its main features.+

The graph has a vertical asymptote at x = 0, crosses the x‑axis at (1,0), and increases slowly to infinity. It passes through (e,1). The domain is (0,∞) and the range is (-∞,∞). The derivative is 1/x, so the slope is always positive but decreases as x increases.

Q2: What happens to |ln x| for 0 < x < 1? Why?+

For x between 0 and 1, ln x is negative, so |ln x| = -ln x. This part of the graph starts at +∞ near x=0⁺ and decreases to 0 at x=1. It's essentially the reflection of the negative portion of ln x across the x‑axis.

Q3: Is |ln x| differentiable at x = 1? Prove it.+

No. Left‑hand derivative (x<1): derivative of -ln x is -1/x, approaching -1 as x→1⁻. Right‑hand derivative (x>1): derivative of ln x is 1/x, approaching +1 as x→1⁺. Since -1 ≠ +1, the derivative does not exist at x=1. The graph has a sharp corner there.

Q4: How would the graph of |ln(x-2)| differ from |ln x|?+

The entire graph shifts 2 units to the right. The vertical asymptote moves to x=2, the sharp corner moves to (3,0), and the shape remains identical. Domain becomes (2,∞).

Q5: What is the derivative of ln x? Why does it explain the graph's shape?+

d/dx(ln x) = 1/x. For x close to 0, 1/x is very large → the graph shoots downward steeply. At x=1, slope is 1. As x→∞, 1/x → 0, so the graph becomes nearly horizontal. This matches the visual: steep near the y‑axis, flat far to the right.

Q6: Why does the log graph never touch the y‑axis?+

Because ln 0 is undefined — logarithms are not defined for zero or negative numbers. As x approaches 0 from the right, ln x tends to -∞, so the y‑axis is a vertical asymptote, approached but never touched.

Q7: What real‑world phenomenon is modeled by a logarithmic curve?+

The Richter scale (earthquake magnitude) is logarithmic. Sound intensity (decibels) and pH (acidity) are also logarithmic. Any situation where equal multiplicative steps are perceived as equal additive changes — like human perception of loudness or brightness — uses a log scale.
